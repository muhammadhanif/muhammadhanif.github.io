Mengirim Notifikasi Gempa Bumi ke Telegram

October 8, 2019 · Muhammad Hanif

Pada kesempatan kali ini saya berbagi hasil karya ngoding yaitu aplikasi sederhana untuk mengirim notifikasi gempa bumi yang terjadi di wilayah Indonesia ke Telegram.

Aplikasi tersebut saya bangun menggunakan bahasa pemrograman Python. Data gempa bumi yang digunakan bersumber dari BMKG (Badan Meteorologi, Klimatologi, dan Geofisika) yang dapat diakses di laman data.bmkg.go.id/gempabumi/.

Instalasi dan Konfigurasi#

Dalam tutorial instalasi dan konfigurasi ini saya menggunakan sistem operasi distro Linux Ubuntu 18.04.2 LTS. Untuk pengguna distro Linux yang lain, silahkan menyesuaikan.

Instalasi git, python, dan pip#

sudo su

apt-get install git python3 python3-pip

Kloning Repositori#

Source code aplikasi saya tempatkan di repositori GitHub muhammadhanif/info-gempa-bumi-telegram.git.

cd /opt/

git clone  https://github.com/muhammadhanif/info-gempa-bumi-telegram.git

Instalasi Pustaka Python#

Aplikasi kirim notifikasi gempa bumi ke Telegram ini membutuhkan pustaka Python sebagai berikut:

  • untangle : pustaka Python untuk parsing dokumen XML.
  • telegram-send : pustaka Python untuk mengirim pesan dan file ke Telegram.
cd /opt/info-gempa-bumi-telegram/

cat requirements.txt

pip3 install -r requirements.txt

Buat Bot Telegram#

Aplikasi ini juga memerlukan Bot Telegram. Untuk membuat bot telegram silahkan menuju ke akun Telegram @botfather. Ikuti instruksi bagaimana membuat bot di akun tersebut.

Jangan lupa untuk mencatat token dari bot yang dibuat tersebut.

Konfigurasi telegram-send#

Setelah membuat bot telegram dan mendapatkan token bot tersebut, kita lanjutkan untuk melakukan konfigurasi telegram-send.

telegram-send --configure

Saat konfigurasi ini anda akan diminta untuk memasukkan token dari bot yang sudah anda buat tadi.

Konfigurasi Cron#

Agar aplikasi dapat mengirim notifikasi gempa bumi secara real time, kita memerlukan bantuan cron.

Salin dan tempel isi file crontab ke crontab.

cd /opt/info-gempa-bumi-telegram/

cat crontab

crontab -e

Jika tidak ada kesalahan dalam instalasi dan konfigurasi, maka aplikasi akan mengirim notifikasi gempa bumi ke telegram apabila terjadi gempa bumi.

Contoh notifikasi yang terkirim ke Telegram:

Gempa Bumi Dirasakan

Gempa Bumi M 5.0+

Semoga bermanfaat.
